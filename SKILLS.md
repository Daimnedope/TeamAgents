# SKILLS.md

## Purpose

A compact catalog of reusable **study** skill cards for educational mini-product development.
For Claude Code automation, see the skill directories under `.claude/skills/` (each contains a
`SKILL.md`). Both layers can be used side by side.

## Initial Skill Set

1. **Problem Framing Skill**
   - Converts broad idea into target problem and value statement.

2. **PRD Structuring Skill**
   - Produces complete PRD with testable acceptance criteria.

3. **MVP Scoping Skill**
   - Cuts low-value work and keeps only delivery-critical features.

4. **Quality Review Skill**
   - Checks requirement coverage, edge cases, and regressions.

5. **Student Demo Skill**
   - Prepares short demo flow for teacher presentation.

## Implemented Skill Files

- `SKILLS/problem-framing.md`
