# Commands

This folder stores reusable command playbooks (markdown command docs).

## Recommended Commands

- `new-feature.md` - PRD to implementation flow
- `review-code.md` - structured review checklist
- `validate.md` - local validation sequence

## Minimum Command Contract

Each command file should include:

1. Goal
2. Inputs
3. Step-by-step execution
4. Expected output artifacts
5. Failure handling
