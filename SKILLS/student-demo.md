# Skill: Student Demo

## Purpose

Prepare a short, teacher-friendly demo flow for project progress checks.

## Inputs

- Implemented scope
- Acceptance criteria status
- Known limitations

## Output

- 3-5 minute demo script
- Feature checklist for demonstration
- Clear next-step statement

## Guardrails

- Demo only implemented functionality.
- Tie each shown step to a PRD criterion.
- Mention one limitation and one next milestone.

## Reusable Prompt

```
Create a 5-minute demo script based on current project state and PRD criteria.
```
